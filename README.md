# README

# アプリケーション名
wotaku

# アプリケーション概要
アニメ、声優について語り合うアプリです

実装機能は以下の通りです

・投稿一覧機能
・新規投稿機能
・投稿機能
・トップページ遷移機能

使用言語
・Ruby
・Ruby on rails
・HTML
・CSS

# アプリケーションのURL
https://wotaku-33037.herokuapp.com/


# 制作背景
好きなアニメ、声優を好きな人同士で語り合えるためです。（別にアニメ以外のものを投稿可）
私自身アニメ、声優が好きということもありこのような機能を持ったアプリケーションがあればとても
便利だと感じて作成しました。
SNSの場などでも語る場は多いと思うのですがアニメ、声優に特化した場というのはあまりないと感じたので作成しました。
